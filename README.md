# react-native-iconly-2

react iconly 2 icons for react-native as font, not svgs

## Installation

```sh
npm install react-native-iconly-2
```

## Usage

```js
import Iconly2 from "react-native-iconly-2";

// ...

const result = await Iconly2.multiply(3, 7);
```
